# GanttFlow Product Overview

GanttFlow is an online Gantt chart collaboration tool with AI-powered project file parsing capabilities.

## Core Purpose

Enable project managers and teams to create, visualize, and share Gantt charts with minimal effort. The tool reduces the barrier to project visualization through AI technology.

## Key Features

- Dual input modes: manual entry + AI file parsing (Excel/Word)
- AI-powered field recognition (tasks, dates, assignees)
- Visual WYSIWYG Gantt chart editor
- One-click sharing with generated links (no registration required)
- Real-time multi-user collaboration
- Export to PNG/PDF/Excel
- 30-second onboarding to professional Gantt charts

## Target Users

Project managers and team collaborators who need quick, professional project visualization without complex setup.
