## 🧪 测试步骤

### 步骤 1：登录系统

1. 访问 http://localhost:5173
2. 如果没有账号，先注册：
   - 点击"注册"
   - 输入邮箱和密码
   - 确认密码
   - 点击"注册"
3. 使用账号登录

### 步骤 2：配置 AI 设置

1. 在首页点击"+ 新建项目"
2. 点击"📤 上传文件"按钮
3. 点击"⚙️ AI 设置"按钮
4. 配置 API：

**使用 OpenRouter**:
```
API 提供商: OpenRouter
API Key: sk-or-v1-xxxxx（您的 OpenRouter Key）
模型: anthropic/claude-3.5-sonnet（推荐）
```

**使用 Anthropic**:
```
API 提供商: Anthropic
API Key: sk-ant-xxxxx（您的 Anthropic Key）
模型: claude-3-5-sonnet-20241022（推荐）
```

5. 点击"保存"

### 步骤 3：上传文件

1. 拖拽测试文件到上传区域，或点击选择文件
2. 支持的格式：
   - Excel: .xlsx, .xls
   - Word: .docx, .doc
3. 文件大小限制：20MB

### 步骤 4：等待 AI 解析

1. 上传后会显示解析进度
2. AI 会自动识别：
   - 任务名称
   - 开始日期
   - 结束日期
   - 负责人
   - 阶段
   - 任务说明
3. 解析时间：通常 5-15 秒

### 步骤 5：确认字段映射

1. 解析完成后，会显示识别结果
2. 检查字段映射是否正确：
   - 任务名称 → 正确的列
   - 日期字段 → 正确的列
   - 负责人 → 正确的列
3. 如果映射不正确，可以手动调整
4. 点击"确认导入"

### 步骤 6：查看导入结果

1. 任务会自动添加到项目中
2. 在任务列表中查看导入的任务
3. 在甘特图中查看时间线
4. 检查数据是否完整准确

## 🐛 常见问题

### Q1: 提示"请先配置 AI 设置"

**原因**: 未配置 API Key

**解决**:
1. 点击"AI 设置"按钮
2. 输入有效的 API Key
3. 选择模型
4. 点击"保存"

### Q2: 提示"API Key 无效"

**原因**: API Key 错误或已过期

**解决**:
1. 检查 API Key 是否正确
2. 确认 API Key 是否有效
3. 检查 API 提供商是否选择正确
4. 重新生成 API Key

### Q3: 解析一直卡在"解析中"

**原因**: 网络问题或 API 服务问题

**解决**:
1. 检查网络连接
2. 检查后端服务是否运行
3. 查看浏览器控制台错误
4. 查看后端日志
5. 重试上传

### Q4: 识别的字段不准确

**原因**: 文件格式不标准或 AI 识别错误

**解决**:
1. 在字段映射界面手动调整
2. 优化文件格式（使用标准表头）
3. 确保日期格式统一
4. 重新上传文件

### Q5: 日期格式错误

**原因**: 文件中的日期格式不标准

**解决**:
1. 使用标准日期格式：YYYY-MM-DD
2. 或使用：2026/04/15
3. 或使用：2026年4月15日
4. 避免使用模糊的日期表达
