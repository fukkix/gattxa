## 🎓 技术要点

### 乐观锁（Optimistic Locking）

**原理**:
- 不锁定数据
- 使用版本号检测冲突
- 冲突时由用户决定如何处理

**优势**:
- 高并发性能好
- 无需数据库锁
- 用户体验流畅

**实现**:
```typescript
// 更新时检查版本号
UPDATE tasks 
SET name = $1, version = version + 1
WHERE id = $2 AND version = $3
```
