# UI 优化更新说明

## 更新内容

### 1. 负责人颜色统一 ✅

**问题**：任务列表中的负责人标签颜色与甘特图进度条颜色不一致

**解决方案**：
- 在 TaskList 组件中添加 `getAssigneeColor` 函数（与 GanttChart 保持一致）
- 负责人标签使用动态背景色和文字颜色
- 颜色通过哈希算法生成，同一负责人始终显示相同颜色

**效果**：
- 侧边栏负责人标签颜色 = 甘特图任务条颜色
- 8种配色方案，自动分配
- 视觉一致性更好，易于识别

---

### 2. 时间颗粒度切换 ✅

**问题**：切换到"周"或"月"视图时，甘特图没有变化

**解决方案**：

#### 日视图（默认）
- 显示：年月 + 日期 + 星期
- 适合：短期项目（1-3个月）
- 精度：每天

#### 周视图
- 显示：年月 + 周数 + 日期范围
- 适合：中期项目（3-6个月）
- 精度：每周
- 特点：
  - 按 ISO 周（周一开始）分组
  - 显示"第X周"和日期范围（如"4/14-4/20"）
  - 周一显示分隔线

#### 月视图
- 显示：年份 + 月份 + 天数
- 适合：长期项目（6个月以上）
- 精度：每月
- 特点：
  - 显示月份名称和天数
  - 年份分隔线更粗
  - 适合宏观规划

**技术实现**：
- 使用 dayjs 的 `isoWeek` 和 `weekOfYear` 插件
- 根据 `ganttConfig.timeScale` 动态调整时间范围和显示
- 自动调整起止日期到周/月边界

### 3. 中国大陆节假日显示 ✅

**问题**：没有显示节假日，不符合中国用户习惯

**解决方案**：

#### 新增功能
- 显示 2026 年中国大陆法定节假日
- 支持调休工作日标记
- 节假日背景色区分

#### 节假日配置（2026年）
- ✅ 元旦：1月1-3日
- ✅ 春节：2月17-23日
- ✅ 清明节：4月4-6日
- ✅ 劳动节：5月1-5日
- ✅ 端午节：6月25-27日
- ✅ 国庆节：10月1-8日

#### 调休工作日
- ✅ 2月15日、2月28日（春节调休）
- ✅ 9月27日、10月10日（国庆调休）

#### 视觉效果
- **节假日**：红色文字 + 淡红色背景
- **周末**：红色文字 + 灰色背景
- **调休工作日**：绿色文字 + 淡绿色背景
- **工作日**：灰色文字 + 白色背景

#### 控制开关
- "显示节假日"开关：控制节假日标记和背景色
- "显示周末"开关：控制周末背景色
- 两个开关独立控制

## 使用说明

### 切换时间颗粒度

1. 在甘特图上方找到"时间粒度"控制
2. 点击"日"、"周"或"月"按钮
3. 甘特图自动重新渲染

**推荐使用场景**：
- **日视图**：1-3个月的项目，需要精确到天
- **周视图**：3-6个月的项目，按周规划
- **月视图**：6个月以上的项目，宏观规划

### 查看节假日

1. 在甘特图上方找到"显示节假日"开关
2. 打开开关，节假日会以红色标记
3. 节假日列会显示淡红色背景
4. 鼠标悬停可查看节假日名称（日视图）

### 识别负责人

- 侧边栏任务列表中的负责人标签颜色
- 与甘特图中对应任务条的颜色一致
- 快速识别同一负责人的所有任务

## 已知问题

### 1. 节假日数据仅支持 2026 年
**影响**：2027 年及以后的项目无法显示节假日
**解决方案**：后续版本添加多年份支持

### 2. 月视图下任务条可能过小
**影响**：长期项目在月视图下任务条很小，难以点击
**解决方案**：添加最小宽度限制，或提供任务列表快速访问

### 3. 周视图的周数可能跨年
**影响**：12月底和1月初的周数显示可能混淆
**解决方案**：使用 ISO 周标准，明确标注年份
