# GanttXa 用户手册

欢迎使用 GanttXa！这是一个简单易用的在线甘特图协作工具。

---

## 快速开始

### 1. 注册账号

1. 访问 GanttXa 首页
2. 点击"注册"按钮
3. 填写邮箱、密码（至少 6 位）
4. 确认密码
5. 点击"注册"完成

### 2. 创建项目

有两种方式创建项目：

**方式一：手动创建**
1. 登录后点击"+ 新建项目"
2. 输入项目名称
3. 开始添加任务

**方式二：AI 解析文件**
1. 点击"📤 上传文件"
2. 拖拽或选择 Excel/Word 文件
3. 配置 AI API Key（首次使用）
4. 等待 AI 解析
5. 确认字段映射
6. 导入任务

---

## 任务管理

### 创建任务

1. 点击"+ 新建任务"按钮
2. 填写任务信息：
   - **任务名称**（必填）
   - **开始日期**（必填）
   - **结束日期**（可选，留空表示进行中）
   - **负责人**（可选）
   - **所属阶段**（选择或自定义）
   - **任务说明**（可选）
   - **依赖任务**（可选，按住 Ctrl/Cmd 多选）
   - **里程碑标记**（可选）
3. 点击"创建"

### 编辑任务

1. 在任务列表中点击任务
2. 修改任务信息
3. 点击"保存"

### 删除任务

1. 在任务列表中找到任务
2. 点击删除图标
3. 确认删除

### 调整任务顺序

1. 鼠标悬停在任务上
2. 拖拽手柄图标出现
3. 拖动任务到目标位置
4. 松开鼠标完成排序

## 甘特图功能

### 查看甘特图

甘特图自动显示所有任务的时间线：
- **任务条**：显示任务的开始和结束时间
- **负责人配色**：不同负责人使用不同颜色
- **里程碑**：用菱形 💎 标记
- **依赖关系**：用虚线箭头连接
- **今日线**：红色竖线标记今天
- **周末**：灰色背景（可关闭）

### 缩放控制

1. 点击 ➖ 缩小（最小 50%）
2. 点击 ➕ 放大（最大 300%）
3. 点击百分比重置为 100%

### 时间粒度

切换不同的时间显示粒度：
- **日**：显示每一天
- **周**：按周显示
- **月**：按月显示

### 快速定位

点击"📍 今天"按钮，快速滚动到当前日期。

### 显示选项

- **显示周末**：开关周末背景显示
- **显示节假日**：开关节假日标记

## 项目协作

### 分享项目

1. 点击"🔗 分享"按钮
2. 选择权限：
   - **仅查看**：只能查看甘特图
   - **可评论**：可以添加评论（M3 功能）
   - **可编辑**：可以编辑任务
3. 选择有效期：
   - 永久有效
   - 1 天 / 7 天 / 30 天 / 90 天
4. 点击"创建分享链接"
5. 链接自动复制到剪贴板

### 管理分享链接

在分享对话框中可以：
- 查看所有分享链接
- 复制链接
- 撤销链接

### 访问分享项目

1. 打开分享链接
2. 无需登录即可查看
3. 根据权限进行操作

## 导出功能

### 导出为图片

1. 点击"📤 导出"按钮
2. 选择"PNG 图片"
3. 点击"导出"
4. 高清图片自动下载

### 导出为表格

1. 点击"📤 导出"按钮
2. 选择"CSV 表格"
3. 点击"导出"
4. 可在 Excel 中打开

### 导出为数据

1. 点击"📤 导出"按钮
2. 选择"JSON 数据"
3. 点击"导出"
4. 完整项目数据备份

## AI 文件解析

### 配置 AI

首次使用需要配置 AI API Key：

1. 点击"上传文件"
2. 点击"⚙️ AI 设置"
3. 选择 API 提供商：
   - **Anthropic**：官方 API
   - **OpenRouter**：第三方聚合
4. 输入 API Key
5. 选择模型（推荐 Claude 3.5 Sonnet）
6. 点击"保存"

### 上传文件

支持的文件格式：
- Excel (.xlsx, .xls)
- Word (.docx, .doc)

文件要求：
- 大小不超过 20MB
- 包含任务相关信息

### 解析流程

1. 上传文件
2. AI 自动识别任务信息
3. 显示解析进度
4. 确认字段映射
5. 导入到项目

### 字段映射

AI 会尝试识别以下字段：
- 任务名称
- 开始日期
- 结束日期
- 负责人
- 阶段
- 说明

如果识别不准确，可以手动调整映射。

## 常见问题

### Q: 如何获取 API Key？

**Anthropic API Key:**
1. 访问 https://console.anthropic.com/
2. 注册账号
3. 在 API Keys 页面创建

**OpenRouter API Key:**
1. 访问 https://openrouter.ai/
2. 注册账号
3. 在 Keys 页面创建

### Q: 任务条为什么没有显示？

检查以下几点：
1. 任务是否有开始日期
2. 开始日期是否在时间范围内
3. 尝试缩放或滚动查看

### Q: 如何删除项目？

1. 在首页找到项目
2. 点击删除图标
3. 确认删除

### Q: 分享链接失效了怎么办？

1. 检查链接是否过期
2. 检查链接是否被撤销
3. 联系项目所有者重新分享

### Q: 如何修改负责人颜色？

负责人颜色是自动分配的，相同负责人始终使用相同颜色。

### Q: 支持移动端吗？

目前优先支持桌面端，移动端体验正在优化中。

## 技巧和建议

### 项目规划

1. **先定义阶段**：将任务按阶段分组
2. **设置里程碑**：标记重要节点
3. **建立依赖**：明确任务之间的关系
4. **分配负责人**：便于追踪和协作

### 时间管理

1. **合理估算**：给任务留出缓冲时间
2. **定期更新**：及时更新任务进度
3. **关注今日线**：了解当前进度
4. **使用里程碑**：分阶段验收

### 团队协作

1. **及时分享**：与团队成员共享项目
2. **设置权限**：根据角色分配权限
3. **定期导出**：备份项目数据
4. **使用评论**：沟通任务细节（M3 功能）
